## CN

Create slice of source array or array-like object.

|参数名|类型|说明|
|-----|----|---|
|array             |array |Array to slice            |
|[start=0]         |number|Start position            |
|[end=array.length]|number|End position, not included|

```javascript
slice([1, 2, 3, 4], 1, 2); // -> [2]
```
