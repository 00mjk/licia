## CN

Check if the value is present in the list.

|参数名|类型|说明|
|-----|----|---|
|array |array object|Target list                         |
|value |*           |Value to check                      |
|return|boolean     |True if value is present in the list|

```javascript
contain([1, 2, 3], 1); // -> true
contain({a: 1, b: 2}, 1); // -> true
```
