## CN

Convert base of a number.

|参数名|类型|说明|
|-----|----|---|
|num   |number string|Number to convert|
|from  |number       |Base from        |
|to    |number       |Base to          |
|返回值|string       |Converted number |

```javascript
convertBase('10', 2, 10); // -> '2'
convertBase('ff', 16, 2); // -> '11111111'
```
