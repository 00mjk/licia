## CN

Simple internationalization library.

### constructor

|Name  |Desc         |
|------|-------------|
|locale|Locale code  |
|langs |Language data|

### set

Add language or append extra keys to existing language.

|Name  |Desc         |
|------|-------------|
|locale|Locale code  |
|lang  |Language data|

### locale

Set default locale.

|Name  |Desc       |
|------|-----------|
|locale|Locale code|

### t

Get translation text.

|Name  |Desc                      |
|------|--------------------------|
|path  |Path of translation to get|
|data  |Data to pass in           |
|return|Translation text          |
